1-Ambiente virtual (Flask: permite criar site com python)
    -Ctrl + Shift + P
    -Escrever "Environment", selecionar "Create Environment", Venv e depois Python

2-Criar site com o flask

3-ender_template: importar arquivo HTML para rodar

4- "!" + "Tab"= Estrutura basica do HTML

5-<Title>: nome da pagina

6-<body>: conteudo da pagina

7-<div>: divisoria

8-id: Único(#) 
-class: Pode haver mais de um item(.)

9-ipconfig: mostra informação do ip de sua internet

10-fazer o deploy para um servidor publico
